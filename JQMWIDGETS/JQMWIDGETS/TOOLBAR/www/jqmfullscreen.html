Introduction

jQuery Mobile is a user interface framework based on jQuery that works across all popular phones, tablets, e-reader, and desktop platforms. Built with accessibility and universal access in mind, we follow progressive enhancement and responsive web design (RWD) principles. HTML5 Markup-driven configuration makes it easy to learn, but a powerful API makes it to deeply cusomize the library.

A page in jQuery Mobile consist of an element with a data-role="page" attribute. Within the "page" container, any valid HTML markup can be used, but for typical pages in jQuery Mobile, the immediate children of a "page" are divs with data-role="header", class="ui-content", and data-role="footer". the baseline requirement fo a page is only the page wrapper to support the navigation system, the rest is optional.

A page can be styled as a dialog that makes the page look like it's a modal style overlay. To give a standard page the apperance of a modal dialog, add the data-rel="dialog" attribute to the link. Transitions can also be set on dialog links.
